## Objectives

1. Analyze historical passenger booking trends  
2. Estimate WTD booking ratios by weekday & product  
3. Build a dashboard to monitor bookings in real-time  
4. Identify performance risk due to poor forecasting  
5. Answer key SQL business questions based on the star schema
